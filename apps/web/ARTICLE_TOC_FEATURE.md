# 文章页面目录功能

## 功能概述

为文章内容页面添加了智能目录导航功能，提升阅读体验。

## 主要特性

### 1. 自动生成目录
- 自动扫描文章中的 h1-h4 标题
- 支持多级标题层次结构
- 自动为标题添加锚点 ID

### 2. 响应式设计

#### 桌面端（≥1280px）
- 目录固定在右侧边栏
- 跟随页面滚动（sticky 定位）
- 白色卡片样式，带圆角和边框
- 显示目录项数量

#### 移动端（<1280px）
- 右下角浮动按钮
- 点击展开侧边栏目录
- 带数字徽章显示目录项数量
- 点击目录项后自动关闭

### 3. 交互功能
- 点击目录项平滑滚动到对应标题
- 滚动时自动高亮当前阅读位置
- 悬停效果和动画过渡
- 支持键盘导航

### 4. 视觉设计
- 渐变色按钮（蓝色系）
- 活动项左侧渐变指示条
- 层级缩进显示
- 文本溢出省略
- 平滑动画过渡

## 文件结构

```
shiyu/apps/web/src/
├── components/
│   └── ArticleToc.vue          # 目录组件
└── views/
    └── ArticleContentView.vue  # 文章内容页（已集成目录）
```

## 使用方法

目录组件已自动集成到文章内容页，无需额外配置。

### 组件参数

```vue
<ArticleToc container-selector=".article-body" />
```

- `container-selector`: 可选，指定扫描标题的容器选择器，默认为 `.article-body`

## 样式定制

目录组件使用 scoped 样式，可以通过以下 CSS 变量进行定制：

- 主色调：`#0ea5e9` (天蓝色)
- 渐变色：`linear-gradient(135deg, #0ea5e9, #0284c7)`
- 背景色：`#ffffff`
- 边框色：`#e2e8f0`

## 浏览器兼容性

- Chrome/Edge 90+
- Firefox 88+
- Safari 14+
- 移动端浏览器

## 性能优化

- 使用防抖处理滚动事件
- 延迟生成目录（300ms）
- 使用 CSS transform 优化动画
- 虚拟滚动支持（未来可扩展）
